在获取`AiISP`工具后，我们就可以简单测试工具了，需要注意这几个基础参数。

## -c, --chip <目标芯片型号>

该参数用来指示当前的芯片型号，填上支持的芯片型号或者使用`auto`自动识别芯片。目前支持以下芯片：

|芯片名称|是否支持|
|:---:|:---:|
|air001|✅|
|air32f103|🔨|

## -p, --port <串口名称>

该参数用来指示是所有的串口名称，如`COM12`、`/dev/ttyUSB2`。

## -b, --baud <串口波特率>

该参数用来表示通信时需要使用的波特率，如若使用不带外置晶振的USB转串口芯片导致通信失败，可能需要降低波特率再试，比如`9600`。
