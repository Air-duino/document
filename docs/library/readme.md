## 支持的外设

目前，AirMCU 通过 Arduino API 支持以下外设。

|外设|Air001|Air32F103|备注|
|:---:|:---:|:---:|:---:|
|ADC|✅|🔨||
|DAC|❌|🔨||
|GPIO|✅|✅||
|I2C|✅|🔨||
|Servo|✅|🔨||
|SPI|✅|🔨||
|PWM|✅|🔨||
|UART|✅|🔨||
|USB|❌|🔨||

- ✅：支持
- 🔨：开发中
- ❌：不支持
